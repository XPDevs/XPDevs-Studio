About

XPDevs Studio is a film studio from the creators of XPDevs.

XPDevs Studio makes films. We use AI to build the code. But every story is written by a person. The characters, the scenes, the dialogue. That all comes from a human. The code is just how we put it on screen.

How We Work

A writer writes the story. The characters, the world, the dialogue. Then we use AI to turn that into a working film. The code does the rendering and the sound. The writer does everything that matters.

Part of XPDevs

XPDevs Studio is part of the XPDevs ecosystem, a group of builders and creators working with AI to make things that were not possible before.

Our Values

Story First

The story decides what we build. Not the other way around. If it does not serve the story, we do not do it.

Human Voice

AI writes the code. People write the characters. That line does not move.

Authentic Worlds

We do the research. If the film is set in the 1700s, it should look and feel like the 1700s. Details matter.

Accessible Cinema

These films run in a browser. No special hardware. No downloads. Just a link.
